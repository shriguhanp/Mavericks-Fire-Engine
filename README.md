## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org) v18+

### 1. Clone & Install

```bash
git clone https://github.com/your-username/fire-calculator.git
cd fire-calculator
npm install
```

### 2. Configure Environment

Create a `.env` file in the project root:

```env
# Required: Your Gemini API key from https://ai.google.dev/
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 3. Firebase Setup (for Save & Login)

1. Create a project at [console.firebase.google.com](https://console.firebase.google.com)
2. Enable **Authentication → Google provider**
3. Enable **Firestore Database**
4. Copy your Firebase config into `src/firebase.ts`

### 4. Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) 🎉

## 🔢 How the Calculation Works

### FIRE Corpus (4% Rule)
```
FIRE Number = (Monthly Expenses × 12 × 25) × (1 + inflation)^years_to_retirement
```

### Portfolio Projection (Monthly Compounding FV Annuity)
```
Corpus = P × (1 + r/12)^n  +  SIP × [((1 + r/12)^n - 1) / (r/12)] × (1 + r/12)
```
Where `P` = current savings, `r` = annual return rate, `n` = months, `SIP` = monthly investment.

### Blended Return
```
Blended Rate = (IndiaReturn × (1 - globalAlloc) + GlobalReturn × globalAlloc)
```

## 🔐 Firestore Security

Documents are protected by user-level rules — users can only read/write their own data:

```
/users/{userId}         → Auth'd user can read/write own profile
/calculations/{userId}  → Auth'd user can read/write own calculations
```
